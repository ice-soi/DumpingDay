# DumpingDay
曜日ごとの予定を管理することができるアプリです。

## 開発言語
 - C#

## 概要
 - 曜日ごとに予定を入力できます。
 - 設定したメモ帳はXMLで保存しています。
 - 予定は曜日ごとに最大4つ設定できます。
